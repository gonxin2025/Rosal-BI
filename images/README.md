# Images

Recursos graficos del proyecto, separados de `frontend/assets/` (que
son los assets que efectivamente carga la aplicacion en el navegador).
Esta carpeta es mas para material de presentacion/documentacion.

- `logo/` -- logo oficial de Rosal BI (`logo-rosal-bi.png`, recortado del
  banner oficial): iglesia + gráfica ascendente + hoja, en navy y verde.
- `mockups/` -- capturas o disenos de referencia de como se ve/deberia
  verse el dashboard.
- `banner/` -- banner oficial completo (`banner-rosal-bi.png`), usado
  como portada del README raíz. Incluye el logo, el nombre del concurso
  y el eslogan del proyecto.
