# Images

Recursos graficos del proyecto, separados de `frontend/assets/` (que
son los assets que efectivamente carga la aplicacion en el navegador).
Esta carpeta es mas para material de presentacion/documentacion.

- `logo/` -- logo del proyecto en distintos tamaños/formatos. Incluye un
  logo placeholder generado automaticamente (circulo navy + hoja verde +
  "R") -- reemplazalo por el logo oficial cuando este definido.
- `mockups/` -- capturas o disenos de referencia de como se ve/deberia
  verse la consola.
- `banner/` -- imagen de portada para el README o para presentaciones.
