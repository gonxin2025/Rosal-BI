# Licencias de terceros

Este proyecto usa las siguientes librerías de código abierto, cargadas
vía CDN (no vendidas ni modificadas) salvo donde se indica:

| Librería | Licencia | Uso |
|---|---|---|
| [Chart.js](https://www.chartjs.org/) | MIT | Gráficas (dona, barras, línea, área polar) en el dashboard |
| [PapaParse](https://www.papaparse.com/) | MIT | Lectura de archivos CSV en el navegador |
| [SheetJS (xlsx.js)](https://sheetjs.com/) | Apache 2.0 | Lectura de archivos Excel/CSV en el navegador (verificador de calidad) |
| [html2canvas](https://html2canvas.hertzen.com/) | MIT | Captura de pantalla para exportar a PDF |
| [jsPDF](https://github.com/parallax/jsPDF) | MIT | Generación de archivos PDF en el navegador |
| [ExcelJS](https://github.com/exceljs/exceljs) | MIT | Escritura de Excel con formato/notas (verificador de calidad) |
