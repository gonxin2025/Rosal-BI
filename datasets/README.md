# Datasets

- `originales/` — archivos tal como se reciben de cada dependencia. **No
  se versionan si contienen datos personales** (ver `.gitignore` en la
  raiz del repo). Antes de subir cualquier archivo aqui, revisalo con
  `frontend/verificador-calidad/`.
- `procesados/` — versiones limpias y/o anonimizadas, listas para
  publicar en datos.gov.co.
- `catalogo.csv` — inventario de todos los datasets del municipio (ver
  columnas documentadas en `docs/datos-abiertos.md`).
