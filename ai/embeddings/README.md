# Embeddings (no usado actualmente)

El flujo actual no usa busqueda semantica ni embeddings -- el Agente de
IA recibe el analisis ya estructurado (JSON) directamente en el prompt,
sin necesitar recuperar contexto de una base vectorial.

Esta carpeta queda preparada por si mas adelante se agrega, por ejemplo,
busqueda semantica sobre el catalogo completo de datasets del municipio
(para que el ciudadano pregunte en lenguaje muy libre y el sistema
encuentre el dataset correcto sin depender de coincidencia de palabras
clave).
