# Dashboard temático: Educación

## Qué dataset buscar en Rosal BI

- **"MEN_ESTADISTICAS_EN_EDUCACION_EN_PREESCOLAR, BÁSICA Y MEDIA_POR_ETC"**
  (Ministerio de Educación Nacional) — usado para el registro de
  "Usos" del proyecto en datos.gov.co. Confirmado existente.
- Buscar también: "deserción escolar", "cobertura neta/bruta",
  "Programa de Alimentación Escolar (PAE)" — por municipio o ETC
  (Entidad Territorial Certificada).

## KPIs y gráficos sugeridos

- Tasa de matrícula o cobertura (KPI de promedio — el sistema detecta
  automáticamente columnas tipo "tasa" y las promedia en vez de
  sumarlas).
- Tendencia por año, con proyección de rango de confianza.
- Distribución por nivel educativo (preescolar/básica/media) o por
  municipio.
- Preguntarle al Narrador Experto: "¿cómo está El Rosal en este tema
  comparado con otros municipios?".
