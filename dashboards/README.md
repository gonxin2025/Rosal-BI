# Dashboards temáticos

Cada subcarpeta corresponde a un tablero especializado por sector,
pensado para reutilizar el mismo motor de análisis de
`frontend/dashboard/` pero con una vista curada para ese tema
específico (KPIs y gráficas relevantes preseleccionadas, en vez de
analizar cualquier dataset genérico).

Ninguno está desarrollado todavía — son carpetas preparadas para cuando
se prioricen. Cada una debería terminar con su propio `index.html` (o
reutilizar la consola general con un dataset precargado) y un
`README.md` explicando qué datasets usa.
