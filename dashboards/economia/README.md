# Dashboard temático: Economía

## Qué dataset buscar en Rosal BI

- **"Presupuesto de Gastos Ordinarios Histórico"** (Contraloría General
  de la República) — identificado como candidato fuerte para
  demostrar el botón de regresión lineal, por tener series largas
  (varios años de "vigencia" fiscal).
- Buscar también: "ejecución presupuestal", "recaudo predial/ICA",
  "regalías" — por municipio o vigencia.

## KPIs y gráficos sugeridos

- Suma de presupuesto/gasto por vigencia (KPI).
- Tendencia por año — este sector es el más propicio para activar el
  botón "📈 Ver con regresión lineal", al tener series fiscales largas.
- Distribución por rubro o dependencia.
