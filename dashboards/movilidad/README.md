# Dashboard temático: Movilidad

## Qué dataset buscar en Rosal BI

- Buscar en datos.gov.co: "accidentalidad vial", "parque automotor",
  "siniestros de tránsito" — por municipio o vía.

## KPIs y gráficos sugeridos

- Total de siniestros/eventos registrados (KPI).
- Distribución por tipo de vehículo o gravedad.
- Tendencia por año.
- Detección de valores atípicos (el motor estadístico ya la incluye
  automáticamente) — útil para identificar puntos/periodos con
  siniestralidad inusual.
